# Data Cleaning and Exploratory Data Analysis with SQL

This project focuses on cleaning and analyzing a dataset using SQL. The goal is to prepare raw data for analysis and extract meaningful insights by applying different SQL techniques.

## Data Cleaning

The first stage of the project involves cleaning the dataset to ensure the data is accurate, consistent, and ready for analysis. Several steps were performed during this process.

Duplicate records were identified and removed to avoid incorrect analysis results. The data was also standardized to ensure consistency across the dataset, such as formatting text fields and correcting inconsistent values.

Missing or null values were handled by populating the data where possible. This helped maintain completeness and made the dataset more reliable for analysis.

Unnecessary columns and rows that were not relevant to the analysis were removed to simplify the dataset and improve query performance.

## Exploratory Data Analysis

After cleaning the data, exploratory data analysis was performed to better understand the dataset and identify patterns.

Various SQL queries were used to summarize and analyze the data. Aggregations, grouping, and filtering helped highlight important trends within the dataset.

Window functions were also used to perform deeper analysis on large datasets. These functions made it easier to calculate rankings, rolling totals, and comparisons across different groups of data.

## Purpose of the Project

The purpose of this project is to practice real-world data preparation and analysis using SQL. Data cleaning is an essential step before any analysis, and this project demonstrates how SQL can be used to transform raw data into a structured format that is easier to analyze.

This project also demonstrates how analytical SQL techniques can be applied to explore large datasets and generate useful insights.
