# Restaurant Operations Analysis

This project analyzes a restaurant dataset using SQL to understand ordering patterns and menu performance. It applies the fundamental SQL concepts I learned, including SELECT, WHERE, GROUP BY, ORDER BY, HAVING, joins, unions, string functions, CASE statements, subqueries, window functions, and CTEs to explore and analyze the data. The goal of the project is to practice applying core SQL techniques on a real dataset and demonstrate how these fundamentals can be used together to perform basic operational analysis.
