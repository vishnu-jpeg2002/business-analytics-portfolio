# SQL Fundamentals

This folder contains my SQL fundamentals practice using MySQL. The purpose is to strengthen core querying skills and understand how different SQL concepts work when analyzing and manipulating data.

The files inside this folder cover essential SQL operations that are commonly used in data analysis and database management. Each script focuses on practicing different query techniques and improving familiarity with writing structured SQL queries.

The concepts practiced in this folder include basic querying with SELECT, filtering data using WHERE, organizing results with ORDER BY, limiting outputs using LIMIT, and creating readable column names through aliasing. It also includes aggregation techniques using GROUP BY and HAVING to summarize data.

More advanced concepts are also practiced, such as combining tables using joins, merging result sets with UNION, and working with text data using string functions. Conditional logic is explored using CASE statements, while subqueries are used to perform nested data operations.

To improve query readability and structure, Common Table Expressions (CTEs) are used. Analytical queries are practiced using window functions to perform calculations across rows of data.

The folder also includes examples of temporary tables for intermediate data processing, as well as database automation concepts such as triggers and scheduled events.

Overall, this folder serves as a collection of SQL exercises aimed at building a strong foundation in database querying and data analysis using MySQL.
