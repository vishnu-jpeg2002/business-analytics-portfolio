# SQL Projects

This folder contains my SQL practice and projects completed while learning and strengthening my SQL skills using MySQL. The work focuses on building a solid understanding of querying, data cleaning, and data analysis by applying SQL concepts to different datasets.

The projects include practicing core SQL fundamentals such as SELECT, WHERE, GROUP BY, ORDER BY, HAVING, joins, unions, string functions, CASE statements, subqueries, window functions, and Common Table Expressions (CTEs). These concepts were used across different exercises to understand how SQL can be applied to retrieve, transform, and analyze data effectively.

In addition to fundamentals, the folder also includes projects focused on data cleaning and exploratory data analysis. These involve tasks such as removing duplicates, standardizing data, handling missing values, and analyzing datasets using SQL techniques to extract meaningful insights.
