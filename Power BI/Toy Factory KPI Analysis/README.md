**Toy Factory Sales Analysis Dashboard (Power BI)**

This project is an end to end Power BI dashboard built using raw CSV datasets from a toy manufacturing business. The goal of the project was to transform raw sales data into meaningful insights through data cleaning, data modeling, DAX calculations, and interactive visualizations.

The dataset consists of four tables: Sales, Products, Stores, and Calendar. The Sales table acts as the fact table containing transactional data such as units sold and revenue, while Products, Stores, and Calendar serve as dimension tables that provide descriptive context for the analysis.

The project began with data preparation in Power Query, where the datasets were cleaned, formatted, and structured properly. After this, a data model was created by establishing relationships between the fact and dimension tables to form a star schema structure.

Using DAX (Data Analysis Expressions), several calculated measures and columns were created to derive meaningful business metrics. These calculations enabled deeper analysis of sales performance across time, products, and store locations.

Finally, an interactive dashboard was developed using Power BI visuals. The report includes KPI cards, bar charts, line charts, slicers, and drill down capabilities to allow users to explore sales performance at different levels.

This project demonstrates the complete Power BI workflow including:

1. Data cleaning and transformation using Power Query

2. Data modeling with fact and dimension tables

3. DAX calculations for business metrics

4. Interactive dashboard creation with multiple visualizations
